Rendering Views

main.go contains all the code which parses the templates (default in web/templates) into a *template.Template. After parsing, all views and components are available in this struct. 

tmpl, err := template.New("").ParseGlob("web/templates/*/*")

Views are then rendered by adding endpoints to the *http.ServeMux, which is provided an HTTP handler. 

mux.HandleFunc("GET /", func(w http.ResponseWriter, r *http.Request) {
  err := tmpl.ExecuteTemplate(w, "index.html", nil)
  if err != nil {
    http.Error(w, err.Error(), http.StatusInternalServerError)
    return
  }
})

Structuring Templates

Pages are composed of *.html views, which include *.tmpl files with reusable blocks of HTML.

Consider the prior example to render views/index.html with head and navbar components in components/head.tmpl and components/navbar.tmpl

index.html 
<head>
  <title>Home</title>
  {{ "{{" }} template "head" . {{ "}}" }}
</head>

<body>
  {{ "{{" }} template "navbar" . {{ "}}" }}
  ... page content ..
</body>
head.tmpl 
{{ "{{" }} define "head" {{ "}}" }}
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<link rel="stylesheet" href="/public/css/custom.css" />
{{ "{{" }} end {{ "}}" }}

This provides the essential requirement of reusing fragments of HTML to compose web pages, without introducing unnecessary complexity.

Reactive Components

An important aspect of hypermedia apps is rendering standalone components retrieved by an HTTP endpoint. Here is an example rendering a bitcoin ticker.

prices.tmpl 
{{ "{{" }} define "prices" {{ "}}" }}
  <div class="col-12 col-md-4 pt-4">
    <div class="card">
      <div class="card-header fw-bold">
        BTC
      </div>
      <div class="card-body">
        {{ "{{" }} .BTC {{ "}}" }}
      </div>
    </div>
  </div>
{{ "{{" }} end {{ "}}" }}

This template gets parsed into the same *template.Template that we use for rendering views.

We can render this template injected with server data as follows:

dashboard.tmpl 
type DashboardApp struct {
  Tmpl *template.Template
}

func (d *DashboardApp) RegisterRoutes(mux *http.ServeMux) {
  mux.HandleFunc("GET /prices", d.renderPrices)
}

type PricesRender struct {
  BTC string
}

func (d *DashboardApp) renderPrices(w http.ResponseWriter, r *http.Request) {
  data := PricesRender{
    BTC: getBTC(),
  }
  err := d.Tmpl.ExecuteTemplate(w, "prices", data)
  if err != nil {
    http.Error(w, err.Error(), http.StatusInternalServerError)
    return
  }
}

Finally, we can use htmx to query this endpoint, and inject the server rendered HTML.

dashboard.html 
<div hx-get="/prices" hx-trigger="every 500ms"></div>

We've now created a live server rendered price ticker that updates every 500ms.
