Introduction

goth-ui is a modern bootstrap theme, alongside a Go backend, with several htmx examples to get you up and running building hypermedia applications.

goth-ui leverages carefully chosen software to prioritize performance, security, scalability, productivity, but most importantly simplicity and ease of use.

With the rapid UI development of Bootstrap, the simplicity of htmx, and the powerful Go standard library, you can not only build fullstack web apps easier than ever, but end with a robust app that will work for decades.
