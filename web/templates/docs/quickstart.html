Quickstart

To get started, first install Go, and make sure it is accessible from the command line. Then, locate goth-ui.zip, and run the following commands. 

> unzip goth-ui.zip
> cd goth-ui

Starting is as simple as: 

> go get
> go run main.go
Listening at localhost:8080

You can then navigate to localhost:8080 in your browser. That's it!

air can also be installed to allow the live reload during development, as shown below. 

> air -- -address 192.168.1.111:8080
> air -- -address 192.168.1.111:8080
web/templates/docs/docbase.html has changed
building...
running...
Listening at 192.168.1.111:8080

Before deployment, make sure to go build main.go. If deploying on a Linux server, I recommend using Caddy to ensure secure HTTPS communication with your deployed app at your registered domain name.
